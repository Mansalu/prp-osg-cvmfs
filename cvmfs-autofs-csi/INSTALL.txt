Quick and dirty install instructions:
=====================================

yum install git go

git clone <URL>
git checkout <branch>
cd prp-osg-cvmfs
mkdir -p /root/go/src/gitlab.cern.ch/cloud-infrastructure
mv $PWD/cvmfs-autofs-csi /root/go/src/gitlab.cern.ch/cloud-infrastructure/cvmfs-csi
cd /root/go/src/gitlab.cern.ch/cloud-infrastructure/cvmfs-csi
make
